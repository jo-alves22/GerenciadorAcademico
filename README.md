# 🎓 Gerenciador Acadêmico

> Sistema web desenvolvido em **Angular** para gerenciamento acadêmico — cadastro e controle de alunos, professores e cursos.

---

## 📌 Sobre o Projeto

O **Gerenciador Acadêmico** centraliza o controle de informações acadêmicas em uma aplicação web moderna e responsiva. A solução permite gerenciar os principais atores do ambiente educacional de forma simples e eficiente.

---

## 🚀 Tecnologias

| Camada | Tecnologia |
|--------|-----------|
| Framework | Angular |
| Linguagem | TypeScript |
| Marcação | HTML5 |
| Estilo | CSS3 |
| Runtime | Node.js |
| Gerenciador de pacotes | NPM |

---

## ✅ Funcionalidades

- Cadastro, listagem, edição e exclusão de **Alunos**
- Cadastro, listagem, edição e exclusão de **Professores**
- Cadastro, listagem, edição e exclusão de **Cursos**

## ⚙️ Como Executar

### Pré-requisitos

- [Node.js](https://nodejs.org/) instalado
- Angular CLI instalado globalmente:

```bash
npm install -g @angular/cli
```

### Passo a passo

```bash
# 1. Clone o repositório
git clone https://github.com/jo-alves22/GerenciadorAcademico.git
```

#### 🖥️ Back-end

```bash
# 2. Acesse a pasta do back-end
cd GerenciadorAcademico/back-end

# 3. Instale as dependências
npm install

# 4. Inicie o servidor
npm start
```

#### 🌐 Front-end

```bash
# 5. Em outro terminal, acesse a pasta do front-end
cd GerenciadorAcademico/front-end

# 6. Instale as dependências
npm install

# 7. Inicie a aplicação
ng serve
```

> ⚠️ **Importante:** inicie o back-end antes do front-end para garantir que a API esteja disponível.

Acesse a aplicação em: [http://localhost:4200](http://localhost:4200)
